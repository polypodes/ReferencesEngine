Bienvenue dans Refaktor !

Refaktor vous permet de gérer vos références, afin de créer rapidement des cahiers sous divers formats. Ainsi, Refaktor vous permet de fournir à votre client un cahier contenant tous les travaux qui peuvent le concerner, sous le support voulu (PDF, présentation, web ...).

Prise en main >

Prise en main

Toutes les actions sont accessibles via le menu latéral. La vue d'ensemble permet d'avoir une vision générale de vos références et de vos cahiers, tandis que la page réglages permet de gérer toutes les informations relatives à votre entreprise. La section projets permet de gérer toutes vos références (qui peuvent être des projets, des produits ...) ainsi que leurs catégories. Pour finir, la section cahier permet de regrouper tous les projets sous différents supports, de les personnaliser et de les exporter/présenter.
