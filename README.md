HTML5 base
==========

Using:
- LESS
- Gulp
	- Compile LESS
	- Minimify CSS
	- Autoprefix CSS
	- Uglify JS
	- Livereload server

Install
=======

- Clone this repo
- Create an alias on Mac OSX :
```
nano ~/.bash_profile
alias html_project='git clone https://github.com/tfarneau/base_html5.git'
```

Don't forget to update your node_module folder.
